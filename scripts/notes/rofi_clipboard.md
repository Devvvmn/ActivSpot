# ✂️ rofi_clipboard.md

## 📝 Описание
Скрипт для управления историей буфера обмена через `rofi` и `cliphist`.

## ⚙️ Логика работы
1. Проверяет наличие запущенного `rofi`. Если запущен — закрывает его.
2. Если не запущен:
   - Получает список записей из `cliphist list`.
   - Передает их в `rofi` для выбора.
   - После выбора декодирует запись через `cliphist decode` и копирует в основной буфер `wl-copy`.

## 🔗 Связи
- **Интеграция:** Работает в связке с `rofi_show.sh` (как часть UI-логики).

## 🛠 Технические детали
- **Тип:** Bash script (One-shot)
- **Зависимости:** `rofi`, `cliphist`, `wl-copy`.
