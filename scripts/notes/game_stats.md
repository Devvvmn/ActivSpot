# 📊 game_stats.sh

## 📝 Описание
Скрипт-демон для сбора системных метрик в реальном времени. Собирает данные о производительности и записывает их в JSON-файл для потребления интерфейсом.

## ⚙️ Логика работы
1. **FPS:** Извлекает данные из логов MangoHud (`/tmp/mangohud/*.csv`).
2. **GPU:** Читает текущую загрузку (`gpu_busy_percent`) и температуру через `sysfs`.
3. **CPU:** Вычисляет загрузку через разницу состояний `/proc/stat` за 0.8с.
4. **RAM:** Анализирует `/proc/meminfo`.
5. **VRAM:** Считывает загрузку видеопамяти через `sysfs`.
6. **Ping:** Проверяет задержку сети через `ping 8.8.8.8`.

## 🔗 Связи
- **Выходные данные:** Сохраняет JSON в `/tmp/qs_game_stats`.
- **Потребляемые данные:** `MangoHud` (логи).
- **Используемый UI:** Визуализируется через QuickShell виджеты.

## 🛠 Технические детали
- **Тип:** Bash script (Polling loop)
- **Зависимости:** `curl`, `awk`, `grep`, `ping`, `mangoHud` (logs).
