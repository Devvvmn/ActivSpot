# 🔍 scan_plugins.sh

## 📝 Описание
Скрипт сканирования директории плагинов для подготовки конфигурации JSON, которую потребляет интерфейс.

## ⚙️ Логика работы
1. Ищет файлы `manifest.json` в подкаталогах `${XDG_CONFIG_HOME}/hypr/plugins/`.
2. Для каждого найденного плагина:
   - Валидирует наличие поля `id`.
   - Inject-ит путь к директории плагина (`pluginDir`) прямо в объект манифеста.
3. Формирует массив JSON.
4. Записывает результат в `~/.cache/quickshell/plugins.json`.
5. Сигнализирует QML-интерфейсу о необходимости обновления через `/tmp/qs_plugins_reload`.

## 🔗 Связи
- **Выходные данные:** `~/.cache/quickshell/plugins.json`.
- **Вызываемые компоненты:** `[[watch_plugins.sh]]`.
- **Потребители:** `QuickShell PluginLoader.qml`.
