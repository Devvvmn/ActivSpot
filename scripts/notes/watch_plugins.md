# 🔌 watch_plugins.sh

## 📝 Описание
Демон мониторинга директории плагинов. Автоматически обновляет состояние системы при добавлении или удалении плагинов.

## ⚙️ Логика работы
1. Следит за директорией `${XDG_CONFIG_HOME}/hypr/plugins/` с помощью `inotifywait`.
2. При любом изменении (создание, удаление, перемещение файла) запускает `scan_plugins.sh`.

## 🔗 Связи
- **Вызываемые компоненты:** `[[scan_plugins.sh]]`.
- **Объект мониторинга:** Директория плагинов Hyprland.

## 🛠 Технические детали
- **Тип:** Bash script (Daemon)
- **Зависимости:** `inotifywait`, `scan_plugins.sh`.
