# 🔈 osd_volume.sh

## 📝 Описание
Скрипт для управления громкостью звука (вверх/вниз/mute) и отправки данных для визуального индикатора (OSD).

## ⚙️ Логика работы
1. Принимает аргументы: `up`, `down`, `mute`, `mic`.
2. Использует `pactl` для изменения уровня громкости или переключения Mute.
3. Записывает текущее состояние (уровень громкости) в `/tmp/qs_osd` для мгновенного обновления UI.

## 🔗 Связи
- **Интеграция:** Вызывается через горячие клавиши Hyprland.
- **Выходные данные:** `/tmp/qs_osd` (для отрисовки виджета).

## 🛠 Технические детали
- **Тип:** Bash script (One-shot)
- **Зависимости:** `pactl`.
