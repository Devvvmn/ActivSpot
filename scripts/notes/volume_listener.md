# 🔊 volume_listener.sh

## 📝 Описание
Демон, отслеживающий изменения уровня громкости в системе через PulseAudio/PipeWire.

## ⚙️ Логика работы
1. Подписывается на события `pactl subscribe`.
2. Фильтрует события изменения громкости (`Event 'change' on sink`).
3. При обнаружении изменения считывает текущий уровень через `pactl get-sink-volume`.
4. Передает данные в `/tmp/qs_osd` для отобрата интерфейсом.

## 🔗 Свя связи
- **Потребители:** Визуальные OSD-виджеты (Dynamic Island).

## 🛠 Технические детали
- **Тип:** Bash script (Daemon / Event-driven)
- **Зависимости:** `pactl` (PulseAudio/PipeWire).
