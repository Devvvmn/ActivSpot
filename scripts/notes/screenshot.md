# 📸 screenshot.sh

## 📝 Описание
Многофункциональный скрипт для создания скриншотов и записи видео с экрана. Поддерживает области (geometry), редактирование и запись звука.

## ⚙️ Логика работы
1. **Режимы:**
   - `[--edit]`: Делает скриншот и открывает `satty` для редактирования.
   - `[--full]`: Скриншот на весь экран.
   - `[--record]`: Запись видео (использует `wl-screenrec`).
2. **Запись видео:**
   - Использует `wl-screenrec` для видеоряда.
   *   Использует `ffmpeg` для захвата аудио (микрофона) в отдельный поток.
   *   В конце склейки (`amix`) объединяет видео и аудио в один `.mp4` файл.
3. **Управление состоянием:** Использует файлы в `~/.cache/qs_recording_state` для управления паузой/возобновлением записи.
