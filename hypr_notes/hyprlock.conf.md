# 🔒 hyprlock.conf

## 📝 Description
Configuration for `hyprlock`, the screen locker. It provides a graphical interface for authentication when the session is locked.

## 🔍 Key Components
- **background**: Visual settings for the lock screen.
- **input-field**: Configuration for the password/auth field.
- **label**: Custom text or clock displays.

## 🔗 Dependencies
- 💤 `hypridle.conf` (Invoked by the idle daemon)
- 🛰️ `hyprland.conf` (Visual environment context)
