# 🖼️ hyprpaper.conf

## 📝 Description
Configuration for `hyprpaper`, a lightweight wallpaper utility for Hyprland.

## 🔍 Key Components
- **Preload**: Loads image files into memory.
- **Wallpaper**: Assigns a loaded image to a specific monitor.

## 🔗 Dependencies
- 🛰️ `hyprland.conf` (Launched as a process)
